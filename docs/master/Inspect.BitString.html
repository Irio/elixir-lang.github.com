inspect(thing, opts)

Represents a string as itself escaping all necessary characters. Binaries that contain non-printable characters are printed using the bitstring syntax.

Examples

iex> inspect("bar")
"\"bar\""
iex> inspect("f\"oo")
"\"f\\\"oo\""
iex> inspect(<<0,1,2>>)
"<<0, 1, 2>>"
