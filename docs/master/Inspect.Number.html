inspect(thing, _opts)

Represents the number as a string.

Floats are represented using the shortened, correctly rounded string that converts to float when read back with binary_to_float/1. This is done via the Erlang implementation of Printing Floating-Point Numbers Quickly and Accurately in Proceedings of the SIGPLAN '96 Conference on Programming Language Design and Implementation.

Examples

iex> inspect(1)
"1"
