Callbacks

__using__

start_link(module())

Should start any connection pooling or supervision and return { :ok, pid } or just :ok if nothing needs to be done. Return { :error, error } if something went wrong.

stop(module())

Should stop any connection pooling or supervision started with start_link/1.

all(module(), Ecto.Query.t())

Should fetch all results from the data store based on the given query.

create(module(), Record.t())

Should store a single new entity in the data store. And return a primary key if one was created for the entity.

update(module(), Record.t())

Should update an entity using the primary key as key.

update_all(module(), Ecto.Query.t(), Keyword.t())

Should update all entities matching the given query with the values given. The query will only have where expressions and a single from expression. Returns the number of affected entities.

delete(module(), Record.t())

Should delete an entity using the primary key as key.

delete_all(module(), Ecto.Query.t())

Should delete all entities matching the given query. The query will only have where expressions and a single from expression. Returns the number of affected entities.
